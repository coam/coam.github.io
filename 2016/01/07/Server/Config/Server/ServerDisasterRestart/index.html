服务器灾备重建应急处理备忘录 - 服务器意外重启

服务器意外重启,检查以下软件:

一、手动重启GPS数据处理程序 MinaGPS.jar

二、MongoDB相关

可以找到目前的进程

ps -ef | grep mongodb

手动重启 MongoDB

sudo mongod -f /etc/mongodb.conf

或者

/etc/init.d/mongodb restart

测试

三、mysql手动连接redis

select redis_servers_set_v2("43.241.222.110",6379);

四、Redis相关

重建Redis运行数据包括 AccountInfoSerial areaSerialList AppShareInfoList

http://www.syam.cc/index/RebuildRedis

五、重启 PHPResque 队列任务服务（参考 MicroApp/cli/READE.ME文件）

sudo nohup php /data/home/yzhang/www/www/applications/plugins/PHPResque/bin/resque &

六、MQTT推送服务器 emqttd 相关

启动 emqttd 推送服务器

cd RunProject
emqttd/bin/emqttd start
